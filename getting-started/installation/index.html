安装

系统要求

  • Python 3.8 或更高版本
  • pip 包管理器
  • 红塔证券量化账号及权限

使用 pip 安装

基础安装

pip install HongTaQuant

验证安装

安装完成后，可以通过以下方式验证：

import HongTaQuant
print(f"HongTaQuant 版本: {HongTaQuant.__version__}")

# 验证主要模块
from HongTaQuant import HongTaData, HostsManager
print("✓ 安装成功！")

或者在命令行中：

python -c "import HongTaQuant; print(HongTaQuant.__version__)"

升级

升级到最新版本：

pip install --upgrade HongTaQuant

卸载

如果需要卸载：

pip uninstall HongTaQuant

常见问题

权限错误

如果安装时遇到权限错误，可以使用 --user 标志：

pip install --user HongTaQuant

依赖冲突

如果遇到依赖冲突，建议使用虚拟环境：

# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate
# Linux/macOS:
source venv/bin/activate

# 安装
pip install HongTaQuant

网络问题

如果下载速度慢，可以使用国内镜像：

pip install -i https://pypi.tuna.tsinghua.edu.cn/simple HongTaQuant
